# 基于git的开发流程

## 1、新建一个开发分支

```git
git branch dev-0.1
```

## 2、切换到开发分支进行开发任务

```git
git checkout dev-0.1
```
  
## 3、发布，合并到发布分支，进行发布任务

```git
git checkout deploy
git merge dev-0.1
git status
```

## 4、完成合并到master

```git
git checkout master
git merge dev-0.1
git status
```

## 5、进行归档

```git
git checkout dev-0.1
git tag archive/dev-0.1 dev-0.1 -m "归档dev-0.1"
git push --tags
```

## 6、删除开发分支

```git
git checkout master
git branch -D dev-0.1
git branch -d -r origin/dev-0.1  # 标记删除
git push origin :dev-0.1   # 完全删除
```
