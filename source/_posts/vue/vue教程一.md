### 安装 vue3



### 安装 vite
```
npm install -g create-vite-app
```

#### 创建项目
```
create-vite-app projectName
```
安装项目依赖
```
cd project
npm install 
npm run dev
```

#### webpack 创建项目的方法
```
git clone https://github.com/vuejs/vue-next-webpack-preview.git project Nameplates
cd projectName
npm install
npm run dev
```

#### Vue-Cli  创建项目的方法
```
npm install -g @vue/Cli
vue create projectName
cd projectName
vue add vue-next
npm run serve
```
